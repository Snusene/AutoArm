# AutoArm - RimWorld 1.6 Mod

## Overview
AutoArm auto-equips your colonists with the best allowed weapons based on outfit policies.

## Features
- Unarmed colonists search for weapons before other tasks; upgrades happen during downtime.
- Supports brawler and hunter logic, weapon quality, DPS, and colonist skill.
- Outfit settings control which weapons are allowed (see Weapons tab under Apparel).
- Forced weapon logic: right-click to force, drop/lose to clear forced.
- Optimized for performance, supports large colonies.
- Compatible with most Simple Sidearms, CE, Infusion2, probably more, and RimWorld DLCs.

## Installation
1. Place in RimWorld/Mods
2. Auto-Sort
